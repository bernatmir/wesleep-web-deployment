WeSleep API Documentation

Complete API reference for integrating sleep intelligence into your healthcare insurance platform

🔗 Authentication

All API requests require authentication using your API key. Include your API key in the request headers:
Authorization: Bearer your-api-key Content-Type: application/json

📥 Wearable Data Ingestion

POST /api/v1/webhooks/wearable/

⏰ Smart Alarm Prediction

GET /api/v1/alarm/predict/{patient_id}
Get the optimal wake-up time for a specific user based on their sleep patterns
// Query Parameters ?target_time=07:30 // Response { "patient_id": "user-123", "target_time": "07:30", "optimal_time": "07:18", "confidence_score": 0.87, "window_start": "07:00", "window_end": "07:30", "reasoning": "User typically enters light sleep phase around 07:15-07:20 based on 7-night history" }
