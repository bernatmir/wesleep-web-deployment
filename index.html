How WeSleep Works

From wearable data to actionable health insights

Data Flow 
1. Wearable Data Ingestion
   POST /api/v1/webhooks/wearable/
   → Raw sleep data stored as JSONB

2. Smart Alarm Prediction
   GET /api/v1/alarm/predict/{patient_id}
   → 30-minute window optimization

3. Weekly User Insights
   GET /api/v1/insights/weekly/{patient_id}
   → Motivational engagement summaries

4. Monthly Anomaly Detection
   GET /api/v1/insights/monthly/{patient_id}
   → Clinical referral alerts

Key Metrics Analyzed

  • Heart Rate Variability (HRV) - Recovery and stress indicator
  • Sleep Phases Distribution - Light, deep, and REM patterns
  • Blood Oxygen (SpO₂) - Apnea detection support
  • Sleep Efficiency - Quality vs time in bed ratio
  • Trend Analysis - Intra-month deterioration patterns
  • AI-Powered Reasoning - Groq LLM with no-diagnosis constraints

Seamless Integration

Get started in minutes with our developer-friendly API

JavaScript 
const WeSleepAPI = require('wesleep-api');

const client = new WeSleepAPI({
  apiKey: 'your-api-key',
  baseURL: 'https://api.wesleep.pro'
});

// Get sleep data for a user
const sleepData = await client.sleep.getUserData({
  userId: 'user-123',
  dateRange: {
    start: '2024-01-01',
    end: '2024-01-31'
  }
});

console.log(sleepData);
